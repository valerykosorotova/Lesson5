Селекторы

A - Селектор по элементу предназначен для выбора элементов по имени тега

#id - Селектор по идентификатору предназначен для выбора элемента по идентификатору (значению атрибута id)

.class - Селектор по классу предназначен для выбора элементов по классу (значению атрибута class)

.class1.class2 - Селектор для элементов, у которых одновременно заданы несколько классов

* - Универсальный селектор (селектор звёздочка) предназначен для выбора всех элементов.

  • A B - Селектор для элемента, который находится где-либо внутри другого элемента

  1. A > B - Дочерний селектор для элемента, который находится непосредственно внутри другого элемента
  2. A + B - Соседний селектор для элемента, который находится первый следом за другим элементом
  3. A ~ B - Родственный селектор для элементов, которые находятся следом за другим элементом

[attr] - Селекторы по атрибуту предназначены для выбора элементов по имени атрибута

[attr='value'] - Селектор по имени и значению атрибута

[attr^='value'] - Селектор по имени атрибута и значению, с которого оно должно начинаться

[attr$='value'] - Селектор по имени атрибута и значению, на которое оно должно заканчиваться

[attr*='value'] - Селектор о указанному атрибуту и значению, которое должно содержать value

[attr~='value'] - Селектр по имени атрибута и значению, которое содержит value отделённое от других с помощью пробела

[attr|='value'] - Селектор по имени атрибута и его значению, которое равно value или начинается со value
